# Empreendedorismo — Imersão Autodidata

> Base do Empretec (ONU/SEBRAE) + módulos práticos de formalização e gestão. Conteúdo para leigos que querem montar uma empresa com conhecimento sólido.

---

## Como usar este material

Este material foi desenhado em três camadas. Você decide até onde quer ir:

| Camada | O que é | Tempo estimado |
|--------|---------|----------------|
| **Panorama** | Uma única página com visão completa de tudo | 20–30 min |
| **Visão Geral do Módulo** | ~800 palavras por módulo, conceito central de cada área | 10–15 min por módulo |
| **Deep-dive** | Documento completo por tópico, com exercícios e checklists | 20–45 min por tópico |

**Recomendação de entrada:** leia o [Panorama Geral](00-Panorama-Geral.md) primeiro. Ele é um mapa: você vai saber o que cada módulo contém e poderá escolher por onde mergulhar.

## Sobre este material

Baseado no programa Empretec, desenvolvido pela ONU (UNCTAD) e operado exclusivamente pelo SEBRAE no Brasil, com extensão de conteúdo operacional para cobrir lacunas práticas do curso original.

Informações legais e tributárias devem ser verificadas nas fontes oficiais (SEBRAE, Receita Federal, Junta Comercial estadual) antes de qualquer decisão.
